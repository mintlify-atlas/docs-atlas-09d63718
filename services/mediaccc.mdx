media.ccc.de is the streaming platform of the Chaos Computer Club (CCC), providing access to thousands of conference recordings, tech talks, and live streams from hacker conferences and events.

## Supported Features

### Content Types

- **Conference Recordings**: Archived talks from past events
- **Live Streams**: Real-time event coverage
- **Lectures and Presentations**: Technical talks and workshops
- **Keynotes**: Featured presentations from events

### Conferences (Channels)

Conferences serve as organizational units:

- Conference information and metadata
- Event schedules and descriptions
- Talk listings by conference
- Year-based organization
- Conference logos and branding

<Info>
In media.ccc.de, "conferences" are similar to channels in other services - they group related content from specific events.
</Info>

### Live Stream Detection

The service automatically identifies live streams:

- Live stream IDs are detected via `MediaCCCParsingHelper.isLiveStreamId()`
- Uses specialized `MediaCCCLiveStreamExtractor` for live content
- Regular recordings use `MediaCCCStreamExtractor`

### Search

- Search across all recordings and talks
- Find content by speaker, title, or topic
- Conference-specific filtering

### Kiosks (Discovery)

media.ccc.de provides multiple discovery feeds:

- **Conferences**: Browse all available conferences
- **Recent**: Recently added recordings (default kiosk)
- **Live**: Currently running live streams

<Note>
The default kiosk is "Recent" which showcases the latest conference recordings and talks.
</Note>

## Special Features

### Technical Content Focus

- Hacker and security conferences

## Channel Tab Handling

media.ccc.de has a simplified channel structure:

- Single channel tab per conference
- Channel tab shows all conference recordings
- Uses `ReadyChannelTabListLinkHandler` for pre-fetched data
- Falls back to `MediaCCCChannelTabExtractor` when needed

## Technical Details

**Base URL**: `https://media.ccc.de`

**Service ID**: 2

**Link Handler Factories**:
- Stream: `MediaCCCStreamLinkHandlerFactory`
- Channel/Conference: `MediaCCCConferenceLinkHandlerFactory`
- Conferences List: `MediaCCCConferencesListLinkHandlerFactory`
- Recent: `MediaCCCRecentListLinkHandlerFactory`
- Live: `MediaCCCLiveListLinkHandlerFactory`

**Special Extractors**:
- Regular streams: `MediaCCCStreamExtractor`
- Live streams: `MediaCCCLiveStreamExtractor`
- Conference kiosk: `MediaCCCConferenceKiosk`
- Recent kiosk: `MediaCCCRecentKiosk`
- Live kiosk: `MediaCCCLiveStreamKiosk`

## Limitations

<Note>
media.ccc.de support has some limitations:

- No comment system (comments not supported)
- No playlist support (returns null)
- No subscription extraction (returns null)
- No search suggestions (returns null)
- Limited to CCC-related content
- Primarily German and English language content
</Note>
